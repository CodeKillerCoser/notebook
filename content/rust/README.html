Rust 学习笔记知识库

Rust 知识库按“基础心智模型 → 语言核心 → 标准库与容器 → 工具链 → 综合案例”的顺序组织。

推荐学习路径

  1. 先看 所有权与借用心智模型，建立所有权、借用、消费的基本判断。
  2. 再看字符串、Option、Deref、容器这些高频主题。
  3. 工具链笔记用于解决 Cargo、rand、VS Code 配置问题。
  4. LeetCode 和项目案例用于把语法落到真实代码里。

体系入口

完整目录见 Rust 学习笔记索引。

维护说明

  • 语言概念放在 01-基础入门/ 和 02-语法进阶/。
  • 工具链和生态配置放在 03-工具链与生态/。
  • LeetCode、小游戏、项目解析等串联多个知识点的内容放在 04-综合案例/。
  • 如果某个主题开始分散，优先新增“完整心智模型”或“全景图”作为聚合页，再链接到细分笔记。
