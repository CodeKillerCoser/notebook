# 低熵航线品牌资源

品牌名称：**低熵航线**  
英文辅助：**Low Entropy Course**  
Slogan：**在复杂系统中，持续构建秩序**

## 设计概念

品牌图形延续附件中的黑白星球、轨道与航线意象，并拆成可复用的站点资源：

- **星球**：个人知识与经验的核心引力源。
- **航线 / 轨道**：在复杂系统中持续探索、复盘与建立秩序的路径。
- **星尘背景**：零散问题、实践片段与长期观察。
- **低熵**：从混乱输入中沉淀结构化笔记。

## 文件说明

- `low-entropy-route-mark.svg`：纯图形标识，适合 favicon、导航小图标、社交头像裁切。
- `low-entropy-route-lockup-light.svg`：浅色主题横版品牌图。
- `low-entropy-route-lockup-dark.svg`：星夜主题横版品牌图。
- `low-entropy-route-lockup-paper.svg`：纸感主题横版品牌图。
- `low-entropy-route-brand.json`：品牌名称、文案、主题色与资源索引。
