# 流程控制

## `loop` — Rust 原生无限循环

```rust
loop {
    println!("永远执行");
}
```

- `loop` 是 Rust 的原生无限循环关键字
- 编译器对 `loop` 有特殊优化——不生成条件判断指令
- `break` 可以跳出循环

## `while true` 为什么不推荐

```rust
// ❌ 编译器会警告: denote infinite loops with `loop { ... }`
while true {
    // ...
}
```

### 原因

- `while true` 每次迭代前都会**对布尔值 `true` 做一次求值**，多此一举
- Rust 编译器建议用 `loop` 表达"我要的就是无限循环"的意思
- `loop` 更明确地传达了程序员的意图

## `break` 可以带出值

这是 `loop` 独有的特性——`while` 和 `for` 不行：

```rust
let x = loop {
    break 42;
};
// x = 42
```

### 猜数字游戏中的应用

```rust
let secret_number = 42;

loop {
    let mut guess = String::new();
    io::stdin().read_line(&mut guess).expect("读取失败");

    let guess: i32 = guess.trim().parse().expect("请输入数字");

    if guess == secret_number {
        println!("You win!");
        break;  // 猜对时退出循环
    }
}
```

## 条件循环：`while`

```rust
let mut n = 3;
while n > 0 {
    println!("{}", n);
    n -= 1;
}
println!("发射！");
```

## 遍历集合：`for`

```rust
let arr = [10, 20, 30];

// 推荐写法
for elem in arr {
    println!("{}", elem);
}

// 范围语法
for i in 1..=5 {    // 包含 5
    println!("{}", i);
}

for i in 1..5 {     // 不包含 5
    println!("{}", i);
}
```

## 选择结构

### `if` 表达式

```rust
let num = 7;

if num < 5 {
    println!("小于 5");
} else if num == 5 {
    println!("等于 5");
} else {
    println!("大于 5");
}
```

### `if` 是表达式（可以赋值）

```rust
let condition = true;
let x = if condition { 5 } else { 6 };
// x = 5
```

> **注意**: Rust 是静态类型语言，`if` 的两个分支必须返回**同一类型**。
