Rust 学习笔记知识库

这套笔记按学习路径重排：先建立所有权和借用的心智模型，再进入字符串、Option、Deref、容器、并发与工具链，最后通过 LeetCode 和项目案例串联。

推荐学习路径

阶段目标重点文档
1. 入门心智模型知道 Rust 为什么总在问“谁拥有、谁借用、谁被消费”。所有权与借用心智模型、Hello World、Cargo 入门
2. 核心语法与标准库掌握日常写题和写小工具最常碰到的类型与 API。Rust 字符串完整心智模型、Option 方法全景图、Deref 与解引用强制转换
3. 容器与算法表达理解 HashMap、数组、切片、Vec 在算法里的选择。HashMap 与容器选择、LeetCode T3、LeetCode T1 Two Sum
4. 工具链与生态能配置 workspace、依赖、编辑器和常用 crate。Cargo Workspace、rand crate 版本、VS Code 配置
5. 综合实践把所有权、容器、字符串、错误处理和并发放到完整代码中看。猜数字游戏、Rust 进程管理器代码深度解析

01 - 基础入门

文档内容
所有权与借用心智模型核心概念消费、借用、克隆、Option<Box<T>> 等新手高频困惑的底层判断框架。
Hello World第一个 Rust 程序，fn main()、println!、直接运行。
Cargo 入门Cargo 常用命令、Cargo.toml、项目结构、workspace 协作。

02 - 语法进阶

文档内容
Rust 字符串完整心智模型聚合入口str、&str、String、UTF-8、转换、遍历、修改、查找、切分、解析、格式化。
String 处理与字符遍历char_indices()、as_bytes()、chars().nth() 的性能与正确性取舍。
HashMap 与容器选择容器HashMap、栈数组、字节视角和字符视角在滑动窗口中的选择。
Option 方法全景图速查检查、取出、借用、映射、替换、组合、转换等方法分类。
Option 常用组合模式借读取值、链式消费、零方法模式匹配、可变借用修改内部值。
Deref 与解引用强制转换为什么 &String 能当 &str，为什么智能指针能调用内部类型方法。
智能指针全解Box、Rc、Arc、RefCell、Mutex、Weak 的选型。
Send / Sync 线程安全标记线程间移动、共享引用安全性，以及复合类型的发送规则。
控制台 I/Ostdin()、read_line()、trim()、parse()。
流程控制loop、while、for、break 带值和选择结构。
match 模式匹配陷阱裸标识符是新绑定，不是外部变量取值；用 guard 做值比较。

03 - 工具链与生态

文档内容
Cargo Workspace多项目 workspace、members、resolver 版本。
rand crate 版本rand 0.7/0.8/0.10+ API 差异。
VS Code 配置rust-analyzer、打开 workspace、代码跳转。

04 - 综合案例

文档内容
LeetCode T1 - Two SumHashMap 查询、插入时机、所有权和索引返回。
LeetCode T2 - 两数相加链表、Option<Box<ListNode>>、借用与消费。
LeetCode T3 - 无重复字符的最长子串滑动窗口、字符串遍历、ASCII 优化和 Unicode 友好版。
LeetCode T6 - Z 字形转换按行模拟、状态维护、字符串拼接。
LeetCode T7 - 整数反转符号保持和溢出提前判断。
LeetCode T8 - 字符串转换整数 atoi状态机、ASCII 字节扫描、边界截断。
猜数字游戏控制台输入、随机数、match、循环和错误处理的综合入门案例。
Rust 进程管理器代码深度解析结构体、智能指针、并发锁、异步运行时、HashMap 和平台抽象的项目级案例。
